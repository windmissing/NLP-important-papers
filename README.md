# 表格说明

- 20%

阅读：title, abstract  
回答问题：要解决什么问题，用了什么方法，作者主要贡献，得到了什么结果  

- 40%

阅读：figure, introduction, conclusion  
回答问题：同上

- 60%

阅读：全文，跳过公式、技巧、术语、定义  

- 80%

阅读：全文

- 100%

阅读：相关文献、代码
